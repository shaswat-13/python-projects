# Python Mini Projects Collection

![Language](https://img.shields.io/badge/Python-Scripting-yellow)


A collection of small Python projects built to strengthen my understanding of programming fundamentals, algorithms, and problem solving.

This repository contains multiple mini-projects implemented while following the book **"50 Small Python Projects"**. As a personal challenge, I built **two projects per day for ~10 days**, focusing on learning core Python concepts through hands-on implementation.

---

## Project Overview

The repository includes implementations across multiple domains:

### Games
Simple CLI-based games that practice control flow and logic.

- Blackjack
- Guess the Number
- Rock Paper Scissors
- Bagels (number deduction game)

### Algorithms
Programs focused on algorithmic thinking.

- Infix → Postfix conversion
- Factor generator
- Multiplication table generator

### Simulations & Probability
Programs demonstrating probability concepts and mathematical patterns.

- Birthday Paradox simulation
- Monty Hall problem
- Collatz sequence

### Cryptography
Basic encryption concepts.

- Caesar Cipher
- Caesar Cipher Hacker (brute-force attack)

### Data Structures
Implementation of fundamental structures.

- Stack
- Queue

### Utilities & Experiments
Small tools and experiments built while exploring Python.

- Bitmap generator
- Romanized Nepali converter
- PyQt experimentation
- Miscellaneous scripts
