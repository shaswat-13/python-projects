# Running Python Scripts in VS Code (python_small_programs Folder)

This document outlines the steps to run Python scripts within the `python_small_programs` folder using Visual Studio Code.

## Run the Python Script

### Method 1: Right-Click and Run

1.  Right-click anywhere within the `<file_name.py>` file in the editor.
2.  Select **"Run Python File in Terminal"**.
3.  A new terminal window (or panel at the bottom of VS Code) should appear, displaying the output of your script.

### Method 2: Using the Run Button

1.  In the top-right corner of the VS Code editor (when `<file_name.py>` is open), locate the green "Run Python File" play button and click it.
2.  The output will typically appear in the "Python" output panel or the integrated terminal.

### Method 3: Using the Terminal Directly

1.  Open the integrated terminal in VS Code by going to **Terminal > New Terminal**.
2.  Navigate to your `python_small_programs` directory using the `cd` command:

    ```bash
    cd Documents/python_small_programs
    ```

3.  Run the script using the `python3` command:

    ```bash
    python3 <file_name.py>
    ```

4.  You should see the output of your script printed in the terminal.
